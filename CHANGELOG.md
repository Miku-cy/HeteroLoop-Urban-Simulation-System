## 版本号规则

本项目使用[语义化版本号](https://semver.org/lang/zh-CN/)：

```
主版本号.次版本号.修订号
MAJOR.MINOR.PATCH
```

- **MAJOR** (主版本): 不兼容的 API 变更
- **MINOR** (次版本): 向后兼容的功能新增
- **PATCH** (修订): 向后兼容的问题修复

### 示例

- `1.0.0` → 首次正式发布
- `1.1.0` → 新增电瓶车系统
- `1.1.1` → 修复碰撞检测 bug
- `2.0.0` → 重大架构重构，不兼容旧版

## 如何使用

### 查看版本更新

访问 [GitHub Releases](https://github.com/Miku-cy/HeteroLoop-Urban-Simulation-System/releases) 查看每个版本的详细信息。

### 版本对比

```bash
# 查看 0.1.0 和 1.0.0 之间的差异
git log 0.1.0..1.0.0 --oneline
```

### 回滚到指定版本

```bash
# 查看所有版本标签
git tag -l

# 切换到指定版本
git checkout 1.0.0
```
