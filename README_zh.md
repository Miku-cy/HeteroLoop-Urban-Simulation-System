# 🌆 异环都市仿真系统

> 基于 Three.js 的实时 3D 城市交通仿真系统

[English Version](./README.md)

## ✨ 功能特性

- 🚗 **车辆仿真** — 实时车辆生成、路径规划与碰撞检测
- 🛵 **电瓶车系统** — 独立的电瓶车交通流仿真
- 🚶 **行人系统** — 行人移动，支持天桥/人行横道
- 🚦 **自适应信号灯** — 根据交通密度 AI 自动调节信号灯时长
- 🏢 **建筑客流统计** — 每栋建筑的实时客流数据
- 🌃 **夜间城市场景** — 沉浸式暗色主题 3D 城市景观
- ⚡ **性能仪表盘** — 实时指标：通行量、等待时间、碰撞次数

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可运行，无需任何构建步骤！

```bash
# 或者本地启动服务
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 🎮 操作说明

| 操作 | 控制方式 |
|------|----------|
| 旋转视角 | 鼠标拖拽 |
| 缩放 | 滚轮 |
| 切换自适应信号灯 | 空格键 |

## 🛠 技术栈

- **Three.js** — 3D 渲染引擎
- **原生 JavaScript** — 零依赖（仅使用 Three.js CDN）
- **HTML5 Canvas** — 实时 HUD 叠加层

## 📊 面板指标说明

| 指标 | 说明 |
|------|------|
| 车辆通行量 | 已通过车辆总数 |
| 电瓶车通行 | 已通过电瓶车总数 |
| 总客流量 | 行人总流量 |
| 车辆等待 | 平均车辆等待时间 |
| 碰撞事故 | 碰撞事件数 |
| 路面车辆/电瓶车/行人 | 当前路面上的活跃实体数 |
