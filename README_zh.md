## ✨ 功能特性

- 🚗 **车辆仿真** — 实时车辆生成、路径规划和碰撞检测，采用物理引擎驱动
- 🛵 **电瓶车系统** — 独立的电瓶车交通流仿真，支持换道行为
- 🚶 **行人系统** — 行人移动，支持天桥/人行横道和寻路 AI
- 🚦 **自适应信号灯** — 根据实时交通密度分析，AI 自动调节信号灯时长
- 🏢 **建筑客流统计** — 每栋建筑的实时客流数据和热力图
- 🌃 **夜间城市场景** — 沉浸式暗色主题 3D 城市景观，支持动态光照
- ⚡ **性能仪表盘** — 实时指标：通行量、等待时间、碰撞次数、实体追踪
- 🎮 **交互控制** — 鼠标/键盘导航，支持实时参数调整

## 🚀 快速开始

### 方式一：直接浏览器打开（推荐）

直接在浏览器中打开 `index.html` — 无需任何构建步骤或依赖！


## 🎮 操作说明

| 操作 | 控制方式 | 说明 |

| 旋转视角 | 鼠标拖拽 | 围绕城市旋转相机 |

## 🎨 技术亮点

### 1. 实时 3D 渲染
- Three.js WebGL 渲染，60 FPS 高性能表现
- 动态光照，支持昼夜循环仿真
- 实例化网格渲染，支持数百个实体同时显示

### 2. 智能交通系统
- BFS 寻路算法，实现最优路径规划
- 边界框碰撞检测，确保交通安全
- 空间分区技术，实时计算交通密度

### 3. 自适应 AI
- 实时交通流量分析
- 动态信号灯时长调整
- 基于学习的优化（未来增强）

### 4. 性能优化
- 对象池技术，减少垃圾回收
- 细节层次（LOD）优化远处物体
- RequestAnimationFrame 流畅动画
